# プリセット：narrative-b（ナラティブ・記者敬体/引用敬体）

ビジネス系・経営者向けメディアのインタビュー記事向け。

---

## 設定

| 項目 | 設定 |
|------|------|
| 記事タイプ | ナラティブ（記者主導） |
| 記者文 | 敬体（〜です・〜ます・〜しています） |
| 引用文 | 敬体（〜です・〜ます） |
| 登壇者表記 | 〜様（外部取材対象）/ 〜氏（一般的な記事表記） |
| 構成方針 | 部分調整（論理優先） |
| 字数目安 | リード150〜200字 ＋ 本文3000〜5000字 |
| 見出し | `##` のみ |

---

## 引用帰属パターン

1. 「〜」と○○様は話します。
2. ○○様はこう語ります。「〜」
3. ○○様によると、〜といいます。
4. 「〜」と○○様は振り返ります。

---

## サンプル（必読）

執筆を開始する前に、必ずサンプルを読んでフォーマットを確認すること。

- `./samples/sample.txt` — 本プリセットの基本形

サンプルに含まれる構成要素（冒頭の企業名・URL・ライター名、記事本体の文体・見出し・引用帰属パターン）をそのまま踏襲する。
