# プリセット：narrative-a（ナラティブ・記者常体/引用敬体）

IT系・技術系メディアのインタビュー記事向け。

---

## 設定

| 項目 | 設定 |
|------|------|
| 記事タイプ | ナラティブ（記者主導） |
| 記者文 | 常体（〜だ・〜である・〜した） |
| 引用文 | 敬体（〜です・〜ます） |
| 登壇者表記 | 〜氏 |
| 構成方針 | 部分調整（論理優先） |
| 字数目安 | リード150〜200字 ＋ 本文3000〜5000字 |
| 見出し | `##` のみ |

---

## 引用帰属パターン

1. 「〜」と○○氏は話す。
2. ○○氏はこう語る。「〜」
3. ○○氏によると、〜という。
4. 「〜」と○○氏は振り返る。

---

## サンプル（必読）

執筆を開始する前に、必ずサンプルを読んでフォーマットを確認すること。

- `./samples/sample.txt` — 本プリセットの基本形

サンプルに含まれる構成要素（冒頭の企業名・URL・ライター名、記事本体の文体・見出し・引用帰属パターン）をそのまま踏襲する。
