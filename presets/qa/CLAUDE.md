# プリセット：qa（Q&A対話形式）

問答形式で構成するインタビュー記事向け。発言量が多い取材・対談に適する。

---

## 設定

| 項目 | 設定 |
|------|------|
| 記事タイプ | Q&A対話形式 |
| 引用率管理 | 適用しない（記事の大部分が発言で構成されるため） |
| 質問表記 | `──` （全角ダッシュ2つ）で開始 |
| 回答表記 | `氏名：` で開始 |
| 引用文 | 敬体（〜です・〜ます）に整形 |
| 字数目安 | 全体3000〜6000字 |

---

## 適用ルール

- A-3（口語整形）・A-4（文末多様化）・A-5（語彙重複回避）は適用する
- 引用率管理（A-2）は適用しない
- インタビュアーの質問は読者向けに整理・再構成してよい
- 専門用語・固有概念の初出時には（）で補足説明を入れる

---

## サンプル（必読）

執筆を開始する前に、必ずサンプルを読んでフォーマットを確認すること。

- `./samples/sample.txt` — 本プリセットの基本形

サンプルに含まれる構成要素（冒頭のメタデータ・＜タイトル案＞・＜リード文＞・本文の質問応答形式・＜プロフィール＞）をそのまま踏襲する。
